開発環境
OS: Windows10
Unity: 2018.2.3f1 (64-bit)

////////////////////////////////////////
使い方(開発環境にて)
Unityにプロジェクト(MoveRobot)をインポートし、実行

MoveRobot/Assets/ReadPath.txt
に記述されたファイルを読み込み、
そこに記述されたソースコードを入力とする

MoveRobot/Assets/ReadPathOB.txt
に記述されたファイルを読み込み、
障害物(BOX),旗（Flag)を設置

指定方法は
Flag,10,3
Box,-4,6

など

///////////////////////////////////////
使い方(実行形式にて)
Unityにプロジェクト(MoveRobot)をインポートし、ビルドを行う

MoveRobot.exeと同じパスのReadPath.txt
に記述されたファイルを読み込み、
そこに記述されたソースコードを入力とする

MoveRobot.exeと同じパスのReadPathOB.txt
に記述されたファイルを読み込み、
障害物(BOX),旗（Flag)を設置

指定方法は
flag,10,3
box,-4,6

など

///////////////////////////////////////
使い方(webブラウザにて)
リポジトリのMoveRobot_WebGLをWebサーバーに置き、MoveRobot_WebGL/index.htmlにアクセス
テキストボックス（左側）に記述されたソースコードを入力とする
テキストボックス（右側）に記述された数値（座標）に障害物(BOX),旗（Flag)を設置

指定方法は
flag,10,3
box,-4,6

など

///////////////////////////////////////
主な対応機能
変数：int,double
条件分岐:if
ループ：for,while
コメント：/* */
ロボット操作：
  モータ：関数に左右の出力を与える（-100から100）
　　　motor(100,100)

センサー各種：
　タッチセンサー：
　　　bumper()
　　　　　正面がぶつかっているとき　１
	ぶつかっていないとき　０　　　を返す

  距離センサー：
     distance()
	正面の距離を返す
　	引数は0:前 1:後ろ 2:左 3:右 方向を表す

　コンパス：
     compass()
	ロボットの向きを返す
　　　　　時計回りに0~360の値を返す

四則演算や条件式に対応
+=や++に対応（ただし、後置、前置に意味はない）
